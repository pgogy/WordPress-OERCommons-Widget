== Description ==

A very basic Widget plugin to display content from OER Commons categorised with the same category as the current post. 

This plugin was developed as part of the World War One Continuations and Beginnings project at the University of Oxford (http://ww1centenary.oucs.ox.ac.uk) 

This plugin is very much in development. Please report problems to http://www.pgogy.com/code/ and http://www.pgogy.com/code/groups/wordpress/oer-commons-widget/

[vimeo https://vimeo.com/60041483]

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
